How to use :

- install : sudo apt install --no-install-recommends ansible
- clone : git clone <this-repo>
- edit configuration : vi ~/ansible/cfg/hosts
- run example : ansible-playbook -i ~/ansible/cfg/hosts ~/ansible/dns/update.yaml
